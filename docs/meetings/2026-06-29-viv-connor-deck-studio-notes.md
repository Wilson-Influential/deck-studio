# Viv and Connor Meeting Notes: Deck Studio

Date: 2026-06-29  
People: Shea, Viv, Connor  
Product: The Studio, especially Deck Studio

## Meeting Overview

The discussion focused on the current progress of Deck Studio, the gaps that would stop wider internal use, and the longer-term idea of a Social Post Studio. The working goal is to make the deck tool "Jane proof": simple enough for non-designers to build useful presentations quickly, while making it difficult to break Influential's brand rules.

## Current Progress

- Brand enforcement is working well. The tool protects core brand choices such as text colour and transparent backgrounds.
- Decks can export to PDF, PNG, share link, and PowerPoint.
- PowerPoint export matters because some users will still want to make advanced layout tweaks after generating a deck.
- Figma and Adobe Illustrator export would be useful for the creative team.
- Users can upload or drop in custom images and move them around the slide.
- Undo and redo are being implemented.
- Alex has been the main tester so far. His real slide-building use has surfaced bugs and missing features.

## Required Deck Builder Additions

- Expand the template range beyond the current general-purpose layouts.
- Add layouts for graphs, costings, project timings, Excel-style data tables, pie charts, bar charts, copy-heavy pages, and case studies.
- Add a team headshot template for three to six people.
- Team headshots should automatically crop into circles so users do not rely on messy Google image pulls or manual formatting.
- Build a more searchable asset library for imagery and custom icons.
- Search examples include functional terms such as "magnifying glass".
- Current library coverage includes large logos, default icons, and photography, but it needs stronger organisation and discovery.

## Social Post Studio Roadmap Idea

- The future Post Studio should support social formats such as Instagram.
- Useful controls would include safe-zone checking and aspect ratio toggles.
- Social posts may work better as a visualizer or inspiration generator than as a rigid template builder, because copy length and image shape vary so much.
- The reference model is closer to Pinterest-style visual inspiration than a locked form.
- The ideal workflow is fast concept generation, then export to Figma, Illustrator, or Canva for manual polish.

## Rollout Strategy

- Keep the first version restricted to the internal Influential brand.
- Do not rush multi-client support until the internal brand version is stable and trusted.
- Once the internal version is solid, explore swapping the brand system so the same engine can support external clients.

## Product Implications

- The next Deck Studio work should prioritise real agency slide patterns over generic template volume.
- The asset library should become a guided brand-safe picker, not just a file bucket.
- Export to PowerPoint is already strategically important. Figma and Illustrator export should be considered for the creative handoff path.
- Social Post Studio should stay adjacent but distinct from Deck Studio. It needs more fluid inspiration tooling, while Deck Studio needs structured presentation logic.
